## problem B

**Note**<br>

In the third test case, during the 15-th second, the clock is already showing water temperature. So you have to wait till the 25-th second, when the clock starts showing time again. You'll wait 25âˆ’15=10 seconds.

```kotlin
fun main() {
```

## problem C

[Firewood](https://codeforces.com/contest/1958/problem/C){:target="_blank"}

It's pretty cold in Berland (yes, even in May). So Monocarp has to light his fireplace.

Monocarp has a big log of wood, which weighs $$2^ğ‘›$$ grams. Monocarp has watched the weather forecast and decided that he has to burn ğ‘˜ grams of wood in the fireplace today, and the remaining $$2^ğ‘›âˆ’ğ‘˜$$ grams of wood will be used tomorrow.

**Input**<br>
The first line contains one integer $$ğ‘¡(1â‰¤ğ‘¡â‰¤10^4)$$ â€” the number of test cases.

Each test case consists of one line containing two integers ğ‘› and ğ‘˜$$(1â‰¤ğ‘›â‰¤60; 1â‰¤ğ‘˜â‰¤2^ğ‘›âˆ’1)$$.

**Output**<br>

**Example**
```
input

2 2
2 1

50 36679020707840

```

**Note**<br>
In the first test case, Monocarp has to cut his log exactly once. Then he will have two logs weighing 2 grams each.

In the second test case, Monocarp has to cut his log of 4 grams once, then cut one of the resulting logs. He will have one log of weight 2 and two logs of weight 1, so he can use two logs to get exactly 3 grams.

**solution 1**
```kotlin
fun main() {
    repeat(readln().toInt()) { println(solve()) }
}

fun solve(): Long {
    var (n, k) = readln().split(' ').map(String::toLong)
    while (k.and(1) == 0L) {
        k = k.shr(1)
        n--
    }
    return n
}
```

## problem D

[Staircase](https://codeforces.com/contest/1958/problem/D){:target="_blank"}

There is a staircase consisting of ğ‘› steps. Each step is either intact, or broken. For each broken step, an integer $$ğ‘_ğ‘–$$ is given denoting the difficulty of repairing it.

You want to repair all broken steps of the staircase, and you want to do it in the minimum possible number of days. What is the minimum total effort it will take to repair all broken steps in the minimum number of days?

**Input**<br>
The first line contains one integer $$ğ‘¡(1â‰¤ğ‘¡â‰¤10^4)$$ â€” the number of test cases.

Each test case consists of two lines:
* the first line contains one integer $$ğ‘›(1â‰¤ğ‘›â‰¤3â‹…10^5)$$ â€” the number of steps;
* the second line contains ğ‘› integers $$ğ‘_1,ğ‘_2,â€¦,ğ‘_ğ‘›(0â‰¤ğ‘_ğ‘–â‰¤10^8)$$. If $$ğ‘_ğ‘–=0$$, then the ğ‘–-th step does not need to be repaired; otherwise, the ğ‘–-th step is broken and ğ‘ğ‘– is the difficulty of repairing it.

**Example**

**Note**<br>

In the second test case, you can repair the 3-rd and the 4-th step during the first day, and the 2-nd step during the second day. The total effort will be 2â‹…(15+8)+13=59.

**solution**
```kotlin
fun main() {
    repeat(readln().toInt()) { println(solve()) }
}
 
fun solve(): Long {
    readln()
    val list = readln().split(' ').map(String::toLong)
    var sum = list.sum().shl(1)
    var max = 0L
    var count = 0
    list.forEach {
        if (it != 0L) {
            if (count.and(1) == 0 && it > max) max = it
            count++
        } else {
            if (count.and(1) != 0) sum -= max
            max = 0L
            count = 0
        }
    }
    if (count.and(1) != 0) sum -= max
    return sum
}
```
